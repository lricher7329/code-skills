# Code Skills Repository

Reusable Claude Code skills for software development tasks.

## Structure

```
code-skills/
├── skills/       # Coding skills
├── agents/       # Coding agents
└── commands/     # Coding commands
```

## Usage

Skills sync to `~/.claude/skills/coding/` via the global sync script.

Project-specific skills should remain in their project `.claude/` folders.
